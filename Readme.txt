# AlexaVolt - Professional Electrical & Photovoltaic Services

This repository hosts the official website for **AlexaVolt**, a specialized provider of electrical installations and solar energy solutions based in Iași, Romania.

## 🏗️ Architecture & Philosophy

In an era of over-engineered web solutions, this project intentionally adopts a **Lean & Static** approach. 

- **Frontend:** Built with [Jekyll](https://jekyllrb.com/) and hosted via **GitHub Pages**.
- **Philosophy:** Performance, security, and zero-maintenance. By choosing a static site generator, we ensure lightning-fast load times and eliminate the attack surface common in CMS platforms.

> **Engineer's Note:** As a DevMLOps Engineer accustomed to orchestrating complex cloud environments—deploying via Helm charts, managing Kubernetes clusters, and configuring Ingress controllers—I chose a static architecture here for its elegant efficiency. This project proves that for high-quality service presentation, a well-configured static stack outperforms unnecessary complexity.
